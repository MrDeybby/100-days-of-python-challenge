100 Days of Python Challenge

Overview

This repository tracks my progress through a 100-day Python challenge. Each day includes a small project, exercise, or set of notes focused on building Python skills through consistent practice.

Repository Structure

  • day01, day02, ...: Daily project folders.
  • days.txt: Daily log/notes.
  • readme: This file.

How to Run

Most days are self-contained. Enter a day folder and run the main script. Example:

cd day01
python main.py

Notes

  • Some days may include extra files like data, templates, or static assets.
  • See each day folder for specific instructions.
